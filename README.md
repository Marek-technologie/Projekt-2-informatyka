Program obsługuje 2 tryby:Auto/Ręczny
Przycisk START rozpoczyna tryb auto, można zatrzymywać i wznawiać proces przyciskiem STOP/WZNOW
Przycisk trzeci zmienia konfigurację z automatycznej na ręczną(odblokowuje przyciski po prawej stronie okna)
Przyciski p1,p2,p3 odpowiadaja za włączanie odpowiadających pomp, przycisk g1 za wlaczenie grzalki
Przycisk RESET zatrzymuje proces i przywraca początkowe ustawienie obiektów
